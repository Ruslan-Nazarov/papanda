*От ИИ - знания, от человека - понимание.*  
*Обучайте не только ИИ, но и человека.*

**papanda** — образовательная среда на основе диалектики, созданная для работы в эпоху искусственного интеллекта. 

Прежние подходы к обучению, требующие бесконечной тренировки памяти и накопления фактов, потеряли смысл — эту функцию теперь выполняет ИИ. Однако генеративные алгоритмы дают лишь готовые ответы и принципиально не способны на осмысление событий или построение картины мира. 

papanda разрешает это противоречие. Приложение берет на себя рутину сбора данных, предоставляя вам готовую методологию для выработки собственного понимания. Вместо пассивного потребления информации, платформа учит вас искать взаимосвязь знаний исключительно через процесс их развития. 

Это единственная комплексная среда, где тайм-менеджмент, изучение языков, точные науки и ведение конспектов неразрывно связаны с методологией развития глубокого диалектического понимания предмета.

## Основные возможности

### 🌀 Диалектика (Ядро системы)
Главный модуль приложения, реализующий ведение конспектов по правилам диалектической логики. Конспект выстраивается не как статический набор фактов, а как процесс развития мысли:
* **Анализ:** Для любого события, знания или формулы определяется простейший элемент и его противоположность.
* **Противоречие:** Проверяется столкновение этих элементов.
* **Разрешение:** Противоречие разрешается — это разрешенное противоречие и становится исходным событием, знанием или формулой, но уже на уровне вашего глубокого понимания.

Модуль усилен двумя специализированными инструментами:
* **AI-поиск противоположностей:** Алгоритм автоматически находит полярные данные в текстах. Вы самостоятельно оцениваете их на противоречивость.
* **Математический парсер:** Радикально упрощает формулы любой сложности, пошагово демонстрируя логику их диалектического развития.

### 📚 Лингвистика и освоение языков
Изучение иностранных языков в papanda — это не механическое заучивание, а инструмент развития диалектического понимания структуры речи:
* **Sentence Trainer:** Обучение языку через структуру предложения. Вы видите, как предложение развивается и разворачивается из своего логического ядра — сказуемого (глагола).
* **Слова в контексте:** При работе со словарем фокус смещается на самостоятельное конструирование смысла. Вы не просто запоминаете перевод, а сразу помещаете новые слова в собственный контекст.
* **Recognitio (Оценка):** Специализированный режим для проверки долгосрочной памяти, позволяющий объективно оценить степень усвоения лексики.

### 📅 Диалектический тайм-менеджмент и самоанализ
Управление временем в papanda — это способ применить принципы диалектики к самому важному: к процессам развития собственной жизни:
* **Фокус на главном (One Thing):** Вместо хаотичного списка дел вы выделяете одну фундаментальную задачу («Главное дело»), из которой логически разворачиваются и ветвятся подчиненные подзадачи. Это позволяет сконцентрировать все ресурсы на главном противоречии текущего этапа, увлекая за собой всё остальное.
* **Хронология и эволюция дел:** Календарь и планирование неразрывно связаны с Хронологией. Она выступает не просто дневником, а инструментом ретроспективного самоанализа, наглядно показывая, как именно развивались и трансформировались ваши дела с течением времени.
* **Сквозные стикеры (Sticky Thoughts):** Проходят через большинство элементов приложения. Стикеры позволяют не просто привязать мгновенные мысли к задачам, событиям или заметкам, но и служат пространством для рефлексии о динамике ваших проектов и жизни.

## 🚀 Установка и запуск

### Вариант 1: Готовый исполняемый файл (.exe)

Скачайте `papanda.exe` из раздела [Releases](https://github.com/your-username/papanda/releases), запустите его — браузер откроется сам.

> ⚠️ Windows может показать предупреждение о неизвестном издателе. Нажмите **«Подробнее» → «Выполнить в любом случае»**.

Все данные (база, настройки, словари) будут автоматически созданы и сохранены в папке `data/` рядом с `.exe` файлом.

Чтобы **остановить сервер**, просто закройте вкладку с приложением в браузере — процесс завершится автоматически.

---

### Вариант 2: Запуск из исходного кода (для разработчиков)

**Требования:**
- Python 3.12+

**Шаги:**

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/your-username/papanda.git
cd papanda

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите приложение
python run_fastapi.py
```

Приложение откроется в браузере автоматически по адресу `http://127.0.0.1:8000`.  
При первом запуске база данных создаётся автоматически и наполняется базовым словарем из встроенного файла `translate.xlsx`.

---

### Сборка .exe самостоятельно

```bash
pip install pyinstaller
```

Запустите файл `create_bundle.bat` в корне проекта. Готовый `papanda.exe` появится в папке `dist/` и будет включать в себя все необходимые шаблоны, статику и базовый словарь.

## 🛠️ Технологический стек

papanda — это полноценное веб-приложение, которое работает локально прямо на вашем компьютере. Никаких внешних серверов, никаких облаков — только вы и ваши данные.

### Backend

| Компонент | Библиотека | Роль |
|---|---|---|
| **Web-фреймворк** | [FastAPI](https://fastapi.tiangolo.com/) | Высокопроизводительный асинхронный фреймворк на Python. |
| **ASGI-сервер** | [Uvicorn](https://www.uvicorn.org/) | Запускает FastAPI-приложение, обеспечивая асинхронную обработку соединений. |
| **Шаблонизатор** | [Jinja2](https://jinja.palletsprojects.com/) | Генерирует HTML-страницы на сервере. |
| **ORM** | [SQLAlchemy](https://www.sqlalchemy.org/) (async) | Описывает модели данных и выполняет запросы к базе данных. |
| **База данных** | [SQLite](https://www.sqlite.org/) + [aiosqlite](https://github.com/omnilib/aiosqlite) | Встраиваемая файловая БД. Все данные хранятся в одном `.db`-файле. |
| **Конфигурация** | [pydantic-settings](https://docs.pydantic.dev/latest/concepts/pydantic_settings/) + [python-dotenv](https://github.com/theskumar/python-dotenv) | Типизированные настройки приложения, загружаемые из `.env`-файла. |
| **Excel** | [openpyxl](https://openpyxl.readthedocs.io/) | Экспорт и импорт словаря слов в формате `.xlsx`. |

### Frontend

- **HTML + Jinja2-шаблоны** — серверный рендеринг страниц с компонентной структурой через `partials/`.
- **Vanilla CSS** — кастомная дизайн-система с переменными, анимациями и glassmorphism-эффектами.
- **Vanilla JavaScript** — интерактивность виджетов, fetch-запросы к API, drag-and-drop, WebSocket-клиенты, динамические обновления без перезагрузки страниц.

### Упаковка и распространение

- **[PyInstaller](https://pyinstaller.org/)** — собирает всё приложение в единый `.exe`-файл, который запускается без установки Python.
