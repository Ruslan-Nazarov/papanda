# История изменений (Changelog)

## 0.6.5 (2026-06-22)
### Исправлено
- Устранено дублирование «Примера конспекта» в списке конспектов. Лишние дубликаты удалены из базы данных.
- Логика загрузки и смены локали для примера конспекта переведена на динамическую подгрузку без необходимости записи/перезаписи в базу данных.
- Убран лишний перезапуск базы данных при смене языка в не-демо режиме.
- Убрано окно дебага (отладки).
- Добавлена полная локализация подсказок (hints) внутри редактора "Диалектика" (BlockManager) на русский, английский и казахский языки.
- Настроены заголовки Content-Security-Policy (CSP) для корректной работы счетчика Яндекс Метрики (добавлен домен `mc.yandex.ru` в разрешенные источники).

## 0.6.4 (2026-06-21)
### Изменено
- Удален PyInstaller и все скрипты локальной сборки (.exe). Приложение полностью переведено на серверную работу (SaaS модель).
- Очищены логики временных папок `_MEIPASS` из `config.py`.
- Удален скрипт завершения сервера при закрытии браузера из `main.py`.
- Обновлен `run_fastapi.py` для корректной локальной разработки.
- Очищен `.gitignore` от мусора сборки PyInstaller.
