# 📚 Smart Notes Feature Reference Handbook

A concise overview of all tools, categories, and AI features available in the interface.

---

## 🧭 Top Toolbar

### 1. Category Selector
Dropdown in the top left allowing you to link the current note to a specific project or domain.

### 2. Note Title
Document title. Automatically saved on change.

### 3. Handbook & Guide (`📖 How it works?`)
* **Handbook** — quick reference for buttons and UI features (this document).
* **How it works?** — detailed methodological guide on dialectical analysis principles.

### 4. Dashboard (`🏠 Dashboard`)
Navigate to the main dashboard panel with notes overview, habits, and analytics.

### 5. Relations (`🔗 Relations`)
Manage relations graph between notes.

### 6. Menu (`⋮ Menu`)
Quick access to creating new notes, loading examples, previous notes history, and AI parsers.
