# Что нового в papanda

## v0.6.13 (июль 2026)

- **Выбор демонстрационного примера** — при клике на кнопку «Пример» в меню обучения теперь открывается окно выбора из двух готовых разборов: «🔺 Теорема Пифагора» и «➕ Суммирование».
- **Интеллектуальный помощник (AI Assistant)** — добавлен глобальный интерактивный ИИ-помощник в отдельном окне диалога.
- **Защита системных конспектов** — многоуровневая защита от удаления базовых демонстрационных конспектов на всех языках.
- **Режим «Свернуть левую колонку»** — новый режим компактно сворачивает левую колонку в иконки-кнопки.
- **Управление счетчиками на дашборде** — клик на виджеты отсчета в шапке открывает окно управления историей и датами.
- **Интерактивная сортировка в оглавлении** — перетаскивайте разделы и главы мышью (Drag & Drop).
- **Интерактивные ссылки (Смарт-чипы)** — связывайте слова и фразы с другими блоками конспекта.
- **Экспорт в Markdown и PDF** — быстрое сохранение конспектов в удобных форматах.

## v0.6.12 (июль 2026)

- **История версий конспектов** — теперь при каждом сохранении создаётся снимок конспекта. Восстановить любую версию можно через кнопку «🕒 История версий» в панели управления.
- **Корзина** — удалённые конспекты теперь попадают в корзину. Их можно восстановить или удалить окончательно через меню «⋮».
- **Комментарии к вопросам** — при выделении непонятного фрагмента текста теперь можно ввести описание проблемы. Комментарий отображается в тултипе при наведении.
- **Формулы через `$` и `$$`** — формулы, обёрнутые в знаки доллара, автоматически распознаются и рендерятся на холсте.
- **Блок цитаты** — в панели форматирования появилась кнопка `"` для создания стильных блоков цитат с подписью автора/источника.
- **Графики по точкам** — во вкладке «Графики» добавлена поддержка точечных координат.
- **Случайный порядок предложений** — в языковом тренажёре предложения теперь перемешиваются при выборе языка.

## v0.6.11 (июнь 2026)

- **Хаки понимания** — в каждом блоке конспекта появилась кнопка «💡» с методическими советами по осмыслению формул и тезисов.
- **Персональный словарь терминов** — к каждому блоку можно прикреплять ключевые термины с определениями.
- **История конспекта** — добавлена кнопка «🕒» для просмотра и восстановления предыдущих версий.
- **Перетаскивание блоков** — блоки теперь можно менять местами, зажав заголовок карточки.
- **Цветовые схемы блоков** — через кнопку «🎨» можно выбрать индивидуальную цветовую схему.
- **Режим «Только заголовки»** — новый переключатель сжимает все блоки до заголовков.
- **Распознавание формул с изображений** — в парсер формул добавлена загрузка картинок и вставка из буфера (Ctrl+V).
- **Наборы активностей (пресеты)** — в виджете «Обзор занятий» появилась возможность создавать и переключать пресеты.
